Single Scale

Single scale visualizations display a genome from a single perspective - one size. However, some single scale visualizations provide a zooming interaction, which allows the user to expand or shrink the genome view.

Multiple Scales

A multiple scale visualization allows for visualizing data from multiple perspectives/sizes at once. This can be useful for something as large as a genome, particularly being able to view individual genes as well as the genome on a whole, as scaling would allow.
